# Tryhackme SOC Level 1
Path: https://tryhackme.com/paths
## Skills Learned
### Log Analysis & Investigation
-Analyzing Windows, Linux, and network logs
- Identifying Indicators of Compromise (IOCs)
- Investigating authentication failures, privilege escalation, and lateral movement
- Detecting malicious command-line activity and suspicious process behavior

### SIEM Operations

- Building and executing SIEM queries
- Reviewing alerts and dashboards
- Correlating events to identify threats
- Working with Splunk, Elastic Stack, and Chronicle-style log analysis

### SOC Analyst Workflows
- Alert triage, case management, and documentation
- Classifying alerts (benign, suspicious, malicious)
- Escalation workflows common in SOC environments
- Simulated real-world SOC scenarios and ticketing practices

### Threat Detection & MITRE ATT&CK
- Understanding attacker tactics & techniques
- Mapping detections and alerts to MITRE ATT&CK
- Recognizing common TTPs: persistence, execution, discovery, defense evasion

### Endpoint & Network Security
- Reviewing EDR-style alerts and process trees
- Using Sysmon logs for endpoint monitoring
- Inspecting PCAPs and network traffic with Wireshark
- Detecting C2 traffic, DNS tunneling, and suspicious network behavior

### Threat Intelligence
- Verifying malicious hashes, IPs, and domains
- Using VirusTotal, AbuseIP, hybrid-analysis, OSINT platforms
- Building basic threat profiles based on indicators

### Tools & Technologies Used
- SIEM: Splunk, Elastic, Chronicle-style tools
- Endpoint Monitoring: Sysmon, Windows Event Logs
- Network Tools: Wireshark, TCPDump, PCAP analysis
- Threat Intel: VirusTotal, AbuseIP, OSINT
